# ewshop
基于学习猿地提供的api开发的商城

## 项目模块
```text
├─public
└─src
├─assets
│  ├─css
│  └─images
├─components
│  ├─common     公共组件
│  └─content    内容组件
├─router        路由模块
├─store         vuex模块
└─views         
    ├─category  分类页面模块
    ├─detail    详情页面模块
    ├─home      首页页面模块
    ├─profile   个人页面模块
    └─shopcart  购物车页面模块
```
